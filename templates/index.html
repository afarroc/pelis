🧩 Funcionalidades principales

Cada sección está diseñada para cubrir una etapa del análisis de URLs

📋

Lista Inteligente

Explora las URLs con paginación, filtros por tipo, país y documento. Busca cualquier término al instante y verifica la disponibilidad de cada enlace con un solo clic.

✅

Verificación en tiempo real

Comprueba si una URL sigue accesible mediante peticiones HTTP (HEAD/GET). Cada resultado se guarda en local para futuras consultas. Re-verifica tantas veces como necesites.

📊

Dashboard Analítico

Gráficos interactivos con Chart.js: distribución por tipo, país, documento y dominio. KPIs de verificaciones y accesos. Todo actualizado en tiempo real desde localStorage.

🔗

Registro de accesos

Cada clic en una URL queda registrado con fecha y hora. Visualiza el historial completo y las URLs más visitadas desde el dashboard.

💾

Persistencia local

Verificaciones, filtros, página actual e historial de accesos se guardan en localStorage. Cierra el navegador y retoma exactamente donde lo dejaste.

📥

Exportación de datos

Descarga la lista filtrada en formato CSV o JSON con un solo clic. Ideal para análisis externos, informes o integración con otras herramientas.

🔄

Verificación por lotes

Verifica todas las URLs visibles en la página actual de una sola vez, con un intervalo controlado para no saturar la red.

🗑️

Limpieza de datos

Borra todas las verificaciones, accesos y filtros guardados con un solo botón y empieza de cero cuando quieras.

🛠️ Stack tecnológico

Python 3 Flask Jinja2 Chart.js JavaScript (Vanilla) localStorage API Fetch API HTML5 CSS3 (Grid/Flexbox) Lumen Database
